== Description ==

Sunrise is an opensource and OOP-based WordPress plugin framework. It was designed to speed up plugin deployment, together with sufficient functionality.

See the documentation at <http://gndev.info/sunrise/>


== Installation ==

Unzip plugin files and upload them under your '/wp-content/plugins/' directory.

Resulted names will be:
  './wp-content/plugins/sunrise/*'

See more docs at <http://gndev.info/sunrise/>


== Upgrade Notice ==

Upgrade normally via your Wordpress admin -> Plugins panel.
